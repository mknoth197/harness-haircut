# Multi-Harness Starter

Starter scaffold for repos using multiple AI providers in one codebase.

## Expected pattern

- Provider-specific harnesses in `/harnesses/*`
- Shared assets in `/shared/*`
- Single mapping config in `/.harness/superchargers.yaml`

## Quick start

```bash
npx --yes degit mknoth197/harness-superchargers/examples/multi-harness-starter my-agent-repo
```
